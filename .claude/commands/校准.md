运行音频功率自动校准（闭环调试 Gain/DRC 直到达标）。

从用户输入中提取通道、阻抗、功率和容差参数，然后执行：

```bash
python .claude/skills/audio-auto-calibration/main.py --channels $ARGUMENTS
```

如果用户没指定具体参数，使用 config.json 中的默认配置直接运行：
```bash
python .claude/skills/audio-auto-calibration/main.py
```

参数提取规则：
- "6R6W" → --impedance 6R --power 6
- "8R8W" → --impedance 8R --power 8
- 容差默认 5%，用户说"容差10%"则 --tolerance 10
- 通道名直接传：HDMI、AV、DTV-DVB、ATV-PAL、USB 等

运行完毕后解读 JSON 输出，告知用户校准结果。如果有 report_path，告知报告位置。
