项目概览

本系统实现了 LG webOS 电视扬声器音频功率的全自动化测试与校准。通过 AI Agent 编排多台硬件设备， 自动完成信号源切换、功率测量、参数计算、固件编译与升级，形成完整的闭环调试流程， 将原本需要工程师数小时手动操作的校准过程缩短至全自动完成。

核心功能模块

🔍

音频功率自检（Self-Check）

只读测量模式，不修改任何参数：

  • 遍历指定信号通道，逐一测量 L/R 功率与 THD+N
  • 自动判定声道平衡和功率是否达标
  • 支持标准信号 + Max Input 两种测试工况
  • 输出 PASS/FAIL 判定 + Excel 详细报告
⚙

音频功率自动校准（Calibration）

闭环迭代调试，自动收敛至目标规格：

  • 阶段一：DRC 预检 — 验证 Max Input 裕量
  • 阶段二：Gain 迭代 — 测量→计算→写入→编译→升级→复测（最多5轮）
  • 阶段三：DRC 回调 — 精调限幅阈值并最终验收
  • 支持多规格批量校准（如 4R5W / 4R3W 连续调试）

技术特点

AI 自然语言驱动

基于 Claude Code Agent，工程师只需用自然语言描述需求（如"自动调试 6R6W HDMI"），系统自动识别意图、解析参数并执行完整流程。

全自动闭环调试

从测量到参数修改、编译、固件升级、复测，形成完整的自动化闭环，无需人工干预。

多功放类型适配

自动检测功放芯片类型（SY6045 步进模式 / WA153814 浮点编码），DRC 计算逻辑自动匹配。

多规格批量校准

支持连续校准多个功放规格（如 4R5W、4R3W），自动切换 SOUND_TYPE、编译升级后依次校准。

标准化 JSON 接口

所有设备适配器统一 JSON 输入输出，便于扩展和集成新设备。

自动生成测试报告

每次自检/校准自动生成 Excel 报告，包含 L/R 功率、THD+N、声道平衡、PASS/FAIL 判定。

使用方式示例

# 自检 — 仅测量，不修改参数
$ 自检 6R6W HDMI AV DTV-DVB

# 自动校准 — 闭环调试直到达标
$ 校准 6R6W HDMI AV DTV-DVB ATV-PAL

# 多规格批量校准
$ 校准 4R5W 4R3W --specs ID_SOUND_xxx ID_SOUND_yyy
